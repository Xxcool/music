# uni-music

#### 介绍
基于uni-app开发的一款关于音乐的移动端项目/小程序

#### 技术栈
vue vuex uni-app

#### 接口文档
网易云音乐公共接口 
https://binaryify.github.io/NeteaseCloudMusicApi/#/

#### 安装教程
git clone 本项目 然后拖动到Hbuild X中 运行查看 移动端/微信小程序(需要将manifest.json中的微信小程序配置中的Appid修改为自己的)
